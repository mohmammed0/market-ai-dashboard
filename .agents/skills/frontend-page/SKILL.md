# frontend-page

## Description
Use this skill when implementing or polishing frontend pages in the Vite app.

## When To Use
- Any page under `frontend/src/pages`
- Reusable UI component work
- API integration and product polish

## Rules
- Keep the current dark product feel.
- Reuse shared components before adding one-off markup.
- Use the shared API helper in `frontend/src/lib/api.js`.
- Ensure loading, error, and empty states are present.
- Prefer focused improvements over page rewrites.
